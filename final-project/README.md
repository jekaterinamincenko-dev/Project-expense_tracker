# Izdevumu izsekotājs
Komandrindas Python programma personīgo izdevumu uzskaitei un analīzei.

Programma ļauj pievienot izdevumus, skatīt tos, filtrēt pēc mēneša,
redzēt kopsavilkumu pa kategorijām un eksportēt datus CSV failā.

## Uzstādīšana
Klonē repozitoriju:

git clone https://github.com/jekaterinamincenko-dev/Project-expense_tracker.git

Palaid programmu:

python expense_tracker/app.py

Nepieciešams Python 3.10+.

## Lietošana
Programma piedāvā izvēlni:

1) Pievienot izdevumu  
2) Parādīt izdevumus  
3) Filtrēt pēc mēneša  
4) Kopsavilkums pa kategorijām  
5) Dzēst izdevumu  
6) Eksportēt CSV  
7) Iziet
