Izveidoju projekta plānu un datu struktūru. Sapratu,
ka labāk sadalīt programmu vairākos moduļos.

Izveidoju storage.py JSON saglabāšanai un app.py ar CLI.
Programma var pievienot un parādīt izdevumus.

## 3. solis — filtrēšana un analīze
Pievienoju filtrēšanu pēc mēneša, kopsavilkumu pa kategorijām
un izdevumu dzēšanu.

un dokumentāciju.
